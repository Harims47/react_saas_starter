# React SaaS Starter Rules

Tech Stack:
- React
- Vite
- Redux Toolkit
- React Router DOM
- Axios
- Bootstrap 5
- Bootstrap Icons

Architecture:
- Reusable SaaS starter template
- Must work for any future project

Folder Structure:
- layouts
- pages
- components
- services
- store
- routes
- hooks
- utils

Rules:
- No inline CSS
- Use Bootstrap utility classes
- Create reusable components
- Follow component-based architecture
- Use functional components only
- Use React hooks
- Use Redux Toolkit for auth state
- Use Axios service layer
- Keep code production ready

Layout Requirements:
- Responsive Sidebar
- Responsive Topbar
- MainLayout
- AuthLayout

Authentication Pages:
- Login
- Forgot Password

Common Components:
- AppButton
- AppInput
- AppCard
- AppLoader

Output:
- Create clean reusable code
- No demo or mock business modules
